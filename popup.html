Green Timer & Stopwatch

The best LeetCode time tracker and habit builder to help you become a coding master.

Basic Features

  • Track time for multiple LeetCode problems
  • Analyze habits with 7-day progress reports
  • Live charts for active and historical data

Advanced Features

  • Auto-Detection: Grabs problem names from LeetCode automatically
  • Account Sync: Backup data with Google/Firefox accounts
  • Matrix Rain: High-contrast green developer theme
  • Export: Download history as CSV

How to use

  1. Pin the green icon to your toolbar
  2. Browse LeetCode problems as usual
  3. Open extension; the name is filled automatically!
  4. Click Add then Start to begin
PROMISE: All data is stored in your browser's Local/Sync storage and never sent to 3rd party servers.

Available for Chrome & Firefox.
